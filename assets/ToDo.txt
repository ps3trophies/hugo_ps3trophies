・シリーズ一覧ページ作成(series_list.html)
フロントマターcategoriesに設定
.Site.Taxonomies.categoriesで全カテゴリ(シリーズ一覧)が取得できるので、シリーズごとにページリンクを表示する

{{ range $termName, $entries := .Site.Taxonomies.categories }}
{{ range $page := $entries.Pages }}

たぶんこれでいけるはず。


・ジャンルごとのタイトル一覧ページを作成(genre_title_list.html)
.Site.Taxonomies.tagsで全タグ(ジャンル)取得できるので、上部でジャンル列挙してページ内リンクに。

{{ range $termName, $entries := .Site.Taxonomies.tags }}
{{ range $page := $entries.Pages }}
一覧はこれで、ジャンルごとに折りたたみにする。
